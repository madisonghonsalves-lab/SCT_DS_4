# SCT_DS_4
# Traffic Accident Data Analysis 🚗📊

## Project Overview

This project focuses on analyzing traffic accident data to identify patterns related to weather conditions, road factors, time of day, and accident hotspots.

The goal is to extract meaningful insights from accident data using Exploratory Data Analysis (EDA) and visualization techniques.


## Dataset

US Accidents Dataset

The dataset contains information about:
- Accident location
- Weather conditions
- Time and date
- Severity
- Road-related factors


## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn


## Analysis Performed

### 1. Time of Day Analysis
Analyzed accident occurrence patterns based on different hours of the day.

### 2. Weather Analysis
Studied how weather conditions affect accident frequency.

### 3. Accident Severity Analysis
Visualized severity levels of accidents.

### 4. Road Factors Analysis
Analyzed factors like:
- Junction
- Traffic Signal
- Crossing
- Stop

### 5. Accident Hotspot Visualization
Identified accident-prone locations using latitude and longitude.

### 6. City-wise Accident Analysis
Found cities with the highest number of accidents.


## Results

The analysis helped identify important patterns and contributing factors behind traffic accidents using data visualization.
